## Стек

- Css3
  - Tailwind Css

- React
  - React-redux

  - React icons
  - React slick
  

<br />

## Инструкция по запуску приложения **локально**

_Приложение использует [Node.js](https://nodejs.org/) для своей работы_

```
$ git clone https://github.com/ildarmjs/movie-app-rtk.git
$ cd movie-app-rtk/
$ npm i
$ npm run dev
```
